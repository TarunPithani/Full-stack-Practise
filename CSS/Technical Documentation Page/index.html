Getting Started

First, install the language runtime from the official website.

Then configure your environment variables.

Use a terminal or an IDE for development.

The compiler is lightweight and fast.

install language-x set PATH=/language-x/bin
Syntax

The syntax is inspired by Python and JavaScript.

Indentation is significant and braces are optional.

Comments start with `#`.

# This is a comment

Functions are defined using the `func` keyword.

  • Variables
  • Loops
  • Conditionals
Examples

Let's look at a simple function.

func greet(name): print("Hello " + name)

Calling this function with `greet("Tarun")` will print "Hello Tarun".

Loops are similar to Python.

for i in range(0, 5): print(i)
